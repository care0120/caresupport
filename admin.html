✅ 方針：「特定のLINEアカウントだけを管理者とする」

実現すること：

  • LINEログイン後、getProfile() でユーザー情報を取得

  • userId または displayName が 事前に登録された管理者と一致するかをチェック

  • 一致すれば admin.html へ遷移、違えばアクセス拒否

✅ 必要な情報

あなたが 管理者として許可したいLINEユーザーの userId を知る必要があります。

例）U1234567890abcdefg1234567890abcdefg

このIDはLINEログイン後に JavaScript で取得できます。テスト用に一時的に displayName（表示名）で判定も可能です。

✅ サンプルコード（管理者判定つき）

html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>管理者ログイン</title>
  <script src="https://static.line-scdn.net/liff/edge/2/sdk.js"></script>
</head>
<body>
  <h2>管理者ログイン</h2>
  <button onclick="loginAsAdmin()">LINEでログイン</button>

  <script>
    const ADMIN_USER_ID = "Uxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"; // ← 管理者のLINE userId をここに記入

    function loginAsAdmin() {
      liff.init({
        liffId: "1657254156-85059NlX"
      }).then(() => {
        if (!liff.isLoggedIn()) {
          liff.login();
        } else {
          liff.getProfile().then(profile => {
            const userId = profile.userId; // 管理者判定用
            const displayName = profile.displayName;

            // 管理者かチェック
            if (userId === ADMIN_USER_ID) {
              alert("管理者認証成功：" + displayName + " さん");
              window.location.href = "admin.html";
            } else {
              alert("アクセス拒否：" + displayName + " さんは管理者ではありません。");
            }
          });
        }
      }).catch(err => {
        console.error("LIFFエラー:", err);
        alert("エラーが発生しました");
      });
    }
  </script>
</body>
</html>
